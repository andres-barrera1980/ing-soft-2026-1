# 🚀 Ingeniería de Software I - Periodo 2026-10

> *"Si un enfoque no produce mejor software más rápido, no es ingeniería."*

Bienvenido al repositorio central de la asignatura **Ingeniería de Software I**. En este espacio, transformamos la teoría en soluciones reales, utilizando métodos sistemáticos, disciplinados y las herramientas de última generación que definen la industria en el año 2026.

---

## 🎯 ¿Qué aprenderemos?

El curso está diseñado para dominar los elementos esenciales de la creación de software profesional, con un enfoque en:

- **Fundamentos Sólidos:** Métodos, técnicas y procesos de ingeniería.
- **Modernidad:** *Vibe Coding*, IDEs Agénticos, DevOps y Low-code.
- **Calidad y Seguridad:** Construcción de software mantenible, seguro y productivo.
- **Colaboración:** Metodologías ágiles (Scrum) y trabajo en equipo de alto desempeño.

## 🏗️ El Proyecto del Curso

Trabajaremos en equipos de **3 estudiantes** para desarrollar un software funcional que resuelva retos reales.

### 🛠️ Stack Tecnológico

- **Lenguaje:** Java 25 (LTS).
- **Interfaz:** JavaFX.
- **Persistencia:** Bases de datos Relacionales y No Relacionales.
- **Seguridad:** Autenticación y gestión multiusuario.
- **Herramientas:** GitHub, Jira, IDEs con soporte de IA.
