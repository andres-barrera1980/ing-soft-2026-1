## Casos de prueba para la operación logaritmo

Para la operación implementada (logaritmo), se diseñaron varios casos de prueba que permiten validar tanto el funcionamiento correcto como el manejo de errores. En primer lugar, se incluyeron casos de éxito utilizando valores positivos válidos, por ejemplo calcular el logaritmo de números como 10 o 100, verificando que el resultado sea el esperado. También se consideraron casos relacionados con rangos extremos de enteros, evaluando el comportamiento del método con valores muy grandes y valores cercanos al mínimo permitido mayor que cero. Finalmente, se incluyeron pruebas para valores prohibidos, como el cálculo del logaritmo de 0 o números negativos; en estos casos se verifica que el sistema lance una excepción (`IllegalArgumentException`) para evitar resultados inválidos.
